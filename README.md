# 📈 台股即時/盤後高勝率選股系統 (Taiwan Stock Quant Screener)

本專案是一個基於 **Python** 開發的「台股量化選股系統」，結合**技術面強勢突破**與**籌碼面三大法人鎖碼**策略。透過 `yfinance` 進行全市場 3000+ 檔股票的極速初篩，再利用 `FinMind` 精準比對外資與投信的籌碼動向，篩選出短期內具有強烈爆發力的強勢個股。

專案提供**現代化 GUI 圖形介面**，支援背景多線程運算、即時進度條與日誌顯示，並提供一鍵匯出 Excel (.xlsx) 及 CSV 功能，適合台股短線、波段交易者進行每日選股分析。

---

## 🖥️ 介面功能特色

* **直觀的圖形介面 (GUI)**：採用扁平化配色與清晰排版，操作極其簡單，開箱即用。
* **背景多線程運算**：選股計算完全在背景執行，介面滑順不卡死、不顯示「沒有回應」。
* **即時日誌與進度條**：進度條（0% 到 100%）與日誌主控台即時同步，讓使用者完全掌握下載與分析進度。
* **互動式樹狀表 (Treeview)**：清晰呈現篩選出的股票代號、名稱、當日收盤價、漲跌幅、成交量以及外資/投信超張數。
* **一鍵匯出 Excel 與 CSV**：支援將選股結果匯出為 Excel (`.xlsx`) 及 CSV 檔案，採用 `utf-8-sig` 編碼，Excel 開啟中文絕不亂碼。

## 📊 核心選股策略條件

系統採用**技術面**與**籌碼面**雙重確認的黃金交叉策略：

1. **條件 A（技術面 - 強勢突破）**
   * 當日收盤價創下過去 20 天的新高（突破波段高點）。
   * 當日成交量大於過去 20 天日平均成交量的 **1.5 倍**（量增價入，實質突破）。
2. **條件 B（技術面 - 均線多頭排列）**
   * 短、中、長期均線黃金交叉並向上發散：**5日均線 > 10日均線 > 20日均線**（MA5 > MA10 > MA20）。
3. **條件 C（籌碼面 - 法人鎖碼）**
   * 外資與投信在過去 3 個交易日中，至少有 **2 天**呈現「同步買超」（兩大主力籌碼鎖死）。
4. **條件 D（流動性濾網 - 排除妖股與殭屍股）**
   * 股價大於 **10 元**（排除低價水餃股）。
   * 過去 20 天日平均成交量大於 **1,000 張**（1,000,000 股，確保流動性充足，資金進出容易）。

## 🛠️ 安裝與使用步驟

### 方法 A：執行 pre-compiled `.exe` 執行檔（推薦，無需安裝 Python）
1. 至本專案下載 [taiwan_stock_screener.exe](taiwan_stock_screener.exe)。
2. 雙擊執行該檔案，系統即會彈出圖形介面視窗。
3. 點選 **「開始篩選股票」**，等待進度條跑滿，即可查看結果並點選 **「匯出 Excel」**。

### 方法 B：直接執行 Python 原始碼（適合開發者）
1. **複製本專案**：
   ```bash
   git clone https://github.com/asia17242/0602.git
   cd 0602
   ```
2. **安裝必備套件**：
   ```bash
   pip install yfinance FinMind pandas numpy openpyxl requests
   ```
3. **啟動 GUI 選股程式**：
   ```bash
   python taiwan_stock_screener_ui.py
   ```
4. **啟動純 CLI 命令列版本**（若不需要視窗介面）：
   ```bash
   python taiwan_stock_screener.py
   ```
